# project

## commit message style

| message  | descripton                    |
| -------- | ----------------------------- |
| feat     | 새로운 기능 추가 관련         |
| fix      | 버그 수정                     |
| test     | 테스트 코드 추가              |
| refactor | 코드 리팩토링                 |
| chore    | 빌드 업무, 패키지 매니저 수정 |
| docs     | 문서 수정                     |
| style    | 코드 formatting               |
